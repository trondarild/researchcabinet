# Arguments

The core philosophical KB. Each entry tracks an argument, its key premises, the strongest objections, and the current state of the dialectic.

## How to Use This Section

- Create a sub-page per argument or position cluster
- Use the Argument Analyst agent to reconstruct and map arguments from texts
- Update entries as the dialectic develops — who has replied to whom, and how

## Argument Entry Template

```markdown
## Argument Name

**Source:** Author, Work, Year (section/page)
**Conclusion:** The claim being argued for

### Premises
1. P1
2. P2
3. Therefore C

### Key Assumptions
- What must be true for this to work
- What is taken for granted

### Objections
| Objection | Source | Status |
|-----------|--------|--------|
| Objection text | Author (Year) | Unanswered / Partially answered / Defused |

### Our Response / Current Position
What we think is right and why.
```

## Active Argument Maps

Create sub-pages here for each argument cluster under investigation. Example structure:

- `arguments/consciousness-problem` — arguments about the hard problem
- `arguments/normativity` — debates about the source of normative authority
- `arguments/personal-identity` — psychological vs. biological continuity views
